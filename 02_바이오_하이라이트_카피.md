# Usherinmaking 프로필 바이오 & 하이라이트 카피 (바로 적용)

> 원칙: 첫 화면에서 **"누가·무엇을·왜 우리에게"**가 즉시 읽혀야 한다. 모든 경로는 **링크트리 → LINE 상담**으로 수렴.

## 1. 프로필 바이오 (3개 안)

> 인스타 바이오는 약 150자 제한. 줄바꿈·이모지로 가독성 확보.

### ✅ 안 A — 안심·소통 강조 (추천)
```
오키나와 웨딩·스냅 포토 📷
🇰🇷 한국인 작가 · 여성 스타일리스트 동행
웨딩｜전촬영｜가족｜커플｜리마인드
말 안 통하는 걱정 없이, 인생사진을 안내합니다
👇 가격·후기·예약 (LINE 상담)
```

### 안 B — 감성·로케이션 강조
```
오키나와에서 남기는 인생사진 🌊
한국인 작가가 한국어로 함께합니다 🇰🇷
웨딩 · 허니문 · 가족 · 솔로웨딩 스냅
📍오키나와 전역 로케이션 촬영
👇 패키지 보기 & LINE 예약
```

### 안 C — 신뢰·후기 강조
```
오키나와 웨딩스냅 | 어셔린메이킹
🇰🇷 한국인 작가 + 여성 스타일리스트
✔ 명확한 패키지 ✔ 실제 후기 ✔ 정성 보정
당신의 오키나와를 사진으로 안내합니다
👇 후기·가격·예약
```

**바이오 운영 팁**
- 이름 필드(검색 노출용)에 키워드 삽입: `오키나와 웨딩스냅 | 어셔린메이킹`
- 링크는 **링크트리 1개**로 통일 (아래 4번 참고)
- 시즌 프로모션 시 마지막 줄만 `🎁 6~7월 얼리버드 진행중`으로 교체

## 3. 하이라이트 5종 (커버 라벨 + 첫 장 카피 + 구성)

> 커버는 통일된 톤(같은 색/폰트)으로 브랜드감 형성. 각 하이라이트 첫 장에 한 문장 요약을 넣어 바로 이해되게.

### ① 💰 가격·패키지
- **첫 장 카피**: "투명한 가격, 비교해보고 결정하세요"
- 구성: 패키지별 시간·컷수·보정 컷수·전달기간 → "자세한 견적은 LINE 상담"

### ② ⭐ 후기
- **첫 장 카피**: "먼저 다녀간 분들의 진짜 이야기"
- 구성: 고객 DM/리뷰 캡처, 비포&애프터, 별점 카드

### ③ 📍 촬영 스팟
- **첫 장 카피**: "오키나와, 어디서 찍을까요?"
- 구성: 만좌모/코우리대교/세소코비치 등 스팟별 대표컷 + 한 줄 특징

### ④ 🎬 비하인드(BTS)
- **첫 장 카피**: "촬영 현장은 이런 분위기예요"
- 구성: 촬영 중 영상, 작가·스타일리스트 모습, 자연스러운 진행 모습

### ⑤ ❓ 자주묻는질문(FAQ)
- **첫 장 카피**: "예약 전 가장 많이 묻는 5가지"
- 구성: 우천 시 / 드레스·메이크업 / 예약 방법 / 보정 기간 / 컷수 (각 1장씩)

## 4. 링크트리(또는 링크 허브) 구성안
```
[어셔린메이킹 · 오키나와 웨딩스냅]
1. 💬 LINE으로 바로 상담하기   ← 최상단 (전환 1순위)
2. 💰 가격·패키지 보기
3. ⭐ 실제 후기 모음
4. 📷 포트폴리오 / 공식 사이트
5. 📍 촬영 스팟 안내
6. 🎁 (시즌) 이달의 프로모션
```

## 5. FAQ 답변 템플릿 (하이라이트·DM 공용)
- **Q. 비가 오면요?** → "우천 시 일정 조율 또는 실내·반실내 로케이션으로 대체합니다. 예약금은 보호돼요."
- **Q. 드레스·메이크업도 되나요?** → "여성 스타일리스트가 동행하며, 드레스·헤어·메이크업 패키지 옵션을 제공합니다."
- **Q. 예약은 어떻게?** → "①LINE 상담 → ②날짜·패키지 확정 → ③예약금 → 촬영. 한국어로 편하게 진행됩니다."
- **Q. 보정은 얼마나 걸려요?** → "셀렉 후 OO주 내 전달드립니다. (패키지별 상이)"
- **Q. 컷수는?** → "패키지별 촬영 컷수와 보정 컷수가 다릅니다. 가격 하이라이트를 확인하세요."

> ※ OO 부분은 실제 운영 기준으로 채워 넣으세요.
